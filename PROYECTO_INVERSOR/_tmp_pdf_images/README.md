# Imagenes temporales extraidas de PDFs

Esta carpeta contiene imagenes generadas durante la extraccion de material desde
PDFs de referencia. No forma parte del firmware ni del diseno electronico fuente.

Uso recomendado:

- Mantenerla como material auxiliar de documentacion.
- Mover imagenes a una carpeta final de activos solo cuando se confirme que se
  usaran en un documento entregable.
- No usarla como fuente unica de informacion tecnica; preferir datasheets,
  esquematicos y BOM.
